## 📦 Installation

### Requirements
- Node.js 16+
- npm or yarn

### Steps

1. **Clone the project**
```bash
git clone https://github.com/mustafakaracuha/photographer-landing-page.git
cd photographer-landing-page
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
```

3. **Start the development server**
```bash
npm run dev
# or
yarn dev
```

4. **Open in browser**
```
http://localhost:5173
```
